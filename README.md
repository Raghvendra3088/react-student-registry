# React Student Registry

A student management interface with a Live Clock utility built with React.js.

## Features
- Student Registry: Add, view, delete students with search
- Live Clock: Real-time HH:MM:SS with retro terminal design

## Tech Stack
React.js, JSX, useState, useEffect, CSS-in-JS
