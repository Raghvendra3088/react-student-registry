# React Student Registry

A full-featured student management interface with a built-in live clock utility.

## Projects

### 🎓 Student Registry (Main)
- Add, view, and delete student records (name, age, course)
- Live search/filter across records
- Color-coded course badges and avatar initials
- Built with React functional components and useState

### 🕐 Live Clock (Utility)
- Real-time HH:MM:SS display updated every second
- Shows day, date, and timezone
- Uses useEffect + setInterval for DOM-free updates

## Tech Stack
- React.js (functional components, hooks)
- JavaScript (ES6+)
- CSS-in-JS (inline styles)

## Run Locally

git clone https://github.com/Raghvendra3088/react-student-registry.git
cd react-student-registry
npm install
npm start
